# MetalSlugClone
기존 게임 MetalSlug를 Unity를 활용해서 클론 코딩한 프로젝트입니다.

## 목차
- 개요
- 게임 설명
- 플레이 영상

## 개요
- 프로젝트 이름: MetalSlugClone
- 개발 인원: 1명
- 개발 기간: 8일 (9/13 ~ 9/20)
- 개발 엔진 및 언어: Unity & C#

## 게임 설명
### 조작
좌, 우 방향키: 이동

상 방향키: 위 바라보기

하 방향키: 앉기

Z: 폭탄

X: 공격

C: 점프
